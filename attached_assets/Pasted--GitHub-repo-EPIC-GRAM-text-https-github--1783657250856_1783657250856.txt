## Как импортировать в Replit

1. Replit → **Create Repl**
2. Выбери **Import from GitHub**
3. Вставь:

```text
https://github.com/EpicStarAi/agent-control
```

4. Если repo private — подключи GitHub аккаунт, где есть доступ.
5. Название Repl:

```text
epicgram-sandbox
```

6. После импорта вставь prompt выше в Replit Agent.

Главное: **Replit используем как онлайн-песочницу UI/оператора**, а не как production Telegram runtime. Production остаётся на VPS `194.163.140.26`.
