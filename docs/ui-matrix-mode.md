# Matrix Mode UI

## Visual language

- Deep black background
- Neon green command accents
- Digital rain and scan-line texture
- Terminal-style bordered panels
- Compact operator dashboard density
- Telegram-like chat navigation and chat window
- AI process cards and memory/log consoles

## Layout

Desktop layout uses four working columns:

1. Navigation, brand, account nodes, safety strip
2. Chat navigation and channel filters
3. Main workspace for chat, agents, accounts, logs, or settings
4. AI-agent process stack, memory console, compact logs

Mobile collapses into stacked panels while preserving the same controls and mocked data.
